Installation¶

This part of the documentation covers the installation of jnanomsg.

Leiningen¶

You can add this library as dependence, putting this on yours :dependencies map on project.clj:

[jnanomsg "0.1.0"]

Maven¶

Add Clojars repository definition to your pom.xml:

<repository>
    <id>clojars.org</id>
    <url>http://clojars.org/repo</url>
</repository>

And then the dependency:

<dependency>
    <groupId>jnanomsg</groupId>
    <artifactId>jnanomsg</artifactId>
    <version>0.1.0</version>
</dependency>
